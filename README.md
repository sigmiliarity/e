# Sigma Bundler 
![icon](icon.svg)

A fast, searchable game bundler with virtual scrolling for browsing large game collections. Built with an array-based implementation for optimal performance.

## Features
- Search games in real-time
- Filter by badges
- Virtual scrolling with Clusterize.js for smooth performance with large datasets
- Peer-to-peer capabilities
